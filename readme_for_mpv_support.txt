To add IAC to work with MPV (a video player) you have to put the iac_sync.lua into the script folder of the mpv
"C:\Users\your_pc\AppData\Roaming\mpv\scripts\iac_sync.lua" it's usually here (for windows)

But that's not all, in the fifth/5 line, change the path to match of where you set iac_stream.exe

Now, into the mpv, when you get in, everything will be the same, but if you press "i", it'll open a window to select the iac file, just so, it'll play with the video (movies are possible to watch)
But remember, to have the iac file, you first must create one

No, mpv won't work fine as a player for audio, only videos that plays fine (for iac)

Some cautions, if you don't want to use the iac file, don't name it the same as the movie, or it'll always load him whenever you skip something (+5 seconds or changing in the search bar)
Another thing, if you change the soundtrack middle movie and then change the point where you watching (+5 seconds or changing in the search bar) it'll go back to the iac audio, pay attention to that
